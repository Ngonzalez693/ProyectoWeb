# Health Manager

## Descripción

Health Manager es una aplicación diseñada para ayudarte a gestionar y mejorar tu estilo de vida a través del seguimiento de ejercicios, alimentación y lugares de interés cercanos. 

## Características

### Pantalla principal
- Muestra las funciones principales de la aplicación.
- Botones para **Crear cuenta** e **Iniciar sesión**.

### Crear cuenta
- Formulario para registrarte como usuario.
- Redirección automática a la sección de inicio de sesión una vez creada la cuenta.

### Iniciar sesión
- Acceso a las funcionalidades de la aplicación tras autenticarse.

### Secciones principales (después de iniciar sesión)

1. **Ejercicios**
   - Añade los ejercicios realizados, el tiempo dedicado y las horas de sueño.
   - Visualiza los datos en:
     - Una tabla con los registros.
     - Una gráfica con las horas de sueño.

2. **Alimentación**
   - Registra lo que comiste, junto con el número de calorías.
   - Consulta alimentos recomendados para mejorar tu dieta.

3. **Lugares**
   - Mapa interactivo con:
     - Parques.
     - Gimnasios.
     - Rutas de senderismo cercanas.

### Funcionalidad adicional
- En todas las pantallas (después de iniciar sesión), se puede abrir un slider para cerrar sesión, lo que redirige a la pantalla principal.

## Requisitos del sistema
- Navegador moderno compatible con las tecnologías web actuales.
- Acceso a Internet para funciones de mapas y registro de datos.
